# Pleiades-infra

## 📜 The Sage's Protocols (Manifest Logic)

To maintain order within the watchtower, all manifests are organised into Functional Tiers.

| Block | Tier | Lore Name | Technical Purpose |
| :--- | :--- | :--- | :--- |
| **`0x`** | **The World** | Lagunica Borders | Core Infra: Namespaces, Networking, and Ingress Gateways. |
| **`1x`** | **The Foundation** | Beatrice's Library | Data & Security: Persistent Volumes, Secrets, and ConfigMaps. |
| **`2x`** | **The Residents** | The Sanctuary | The Apps: Deployments, Services, and Database Seed Jobs. |
| **`3x`** | **The Watchers** | The Observatory | Observability: Prometheus, Grafana, and Health Metrics. |

---

### Tower Floors (Namespaces)

* **`lagunica-core`**: The citadel. Cluster-wide tools.
* **`sanctuary-dev`**: The trials. For testing.
* **`pleiades-prod`**: The watchtower. Isolated stable environment.
